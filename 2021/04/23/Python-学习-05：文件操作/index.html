打开文件

文件 (file) 也是一个对象，所有对文件的操作都是基于对这个对象的操作。


打开文件：

tips :

  1. 建议使用原始字符串

  2. 文件路径可以为绝对路径，也可以为相对路径

  3. ‘模式’ 是指打开文件的方式，在后文会详细解释

  4. 一定要在使用完后关闭文件


使用 with 打开文件：

tips :

  1. 注意缩进

  2. 不需要关闭文件


文件打开模式

其它操作

  1. file.read(size) : 从文件读取指定的字节数，未给定或为负则读取所有

  2. file.readline(size) : 读取整行或该行指定的字节数，包括 \n

  3. file.readlines() : 读取所有行并返回一个列表

  4. file.seek(offset[, whence]) : offset 代表需要移动偏移的字节数，如果是负数表示从倒数第几位开始，whence 默认值为 0，给 offset 参数一个定义，表示要从哪个位置开始偏移：0 代表从文件开头算起，1 代表从当前位置开始算起，2 代表从文件末尾算起。

  5. file.tell() : 指针的位置，以字符为单位


OS package

  • os.listdir(path) : 返回路径下所有文件的文件名

  • os.path.splitext(fileName) : 将文件名和扩展名分开

  • os.path.split(path) : 返回文件的路径和文件名
