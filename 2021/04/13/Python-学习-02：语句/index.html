in，not in


位运算符

&，|，^，~，<<，>>


赋值运算符

=

链式赋值

身份运算符

if 条件语句


pass 语句：空语句，是为了保持程序结构的完整性。pass 不做任何事情，一般用做占位语句。

三元操作符：A = X if condition else Y


for 循环


range(begin, end, step) 函数：返回一个以 step 为步长从 begin 到 end 左闭右开的序列。


while 循环


while 和 for 的区别：for 用来遍历，while 用来判断条件。
